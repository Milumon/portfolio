# **App Name**: Milumon Portfolio Split

## Core Features:

- Split-Screen Layout: Implement a split-screen layout, one side for content creation/gaming and the other for web development.
- Animated Social Media Panel: Display an animated panel showcasing social media links (Instagram, TikTok, YouTube) on the content creator side.
- Highlighted Projects Carousel: Display a carousel with highlighted clips, reels, or streams on the content creator side, and highlighted projects with links to GitHub/demo on the web developer side.
- Animated Tech Stack List: Show an animated list of web development technologies (e.g., Next.js, Tailwind) on the web developer side.
- Downloadable CV & Contact Button: Include a 'Download CV' and 'Contact Me' button with a neon glass effect on the web developer side.
- Background Generation: Utilize AI to generate personalized room backgrounds that adapt to the user's individual tastes.

## Style Guidelines:

- Content Creator Side: Vibrant purples (#9400D3), blues (#007BFF), and pinks (#FF69B4) reminiscent of LED lights.
- Web Developer Side: Neutrals, white (#FFFFFF), light grays (#D3D3D3), light blues (#ADD8E6), and plant green (#98FB98).
- Central Border: Use a glassmorphism effect with a glow to create a soft, diffused visual transition. Background color is very pale violet (#F0E8FF, HSL 262, 40, 95).
- Font: 'Inter', a grotesque sans-serif with a modern, machined, objective, neutral look for headlines and body text.
- Desktop: Split-screen with a central border having a softened glass effect. Mobile: Stacked vertically with animated transitions.
- Use Framer Motion for panels and interactive elements. Incorporate parallax and subtle object movement. Employ animated SVG wireframes on the developer side.
- Incorporate relevant icons related to the social media accounts on the left panel (TikTok, Instagram, YouTube) and web development skills in the project cards.
